vscode编译调试c语言

  • 安装插件”C/C++” “Code Runner”
  • 创建一个文件夹作为工作路径
  • 用vscode打开该文件夹（必须是文件夹，只打开单个文件不能调试）
  • 在File->Preferences->settings->User->Extensions->Run Code configuration里勾上”Run In Terminal”
  • 新建一个helloWorld.c文件，随意设一个断点
  • 在debug里按调试会新建一个launch.json文件（在.vscode隐藏文件夹里） 接着会提示创建一个task.json文件，此时已经能够进行运行及调试了（有其他需求可以看着改这两个文件）
